# Global Airline Distribution Analysis

##  Project Overview
This data product analyzes a global airlines dataset to identify which countries have the most active airlines. It helps industry researchers understand the competitive landscape of the aviation market.

## Tech Stack
- **Language:** Python
- **Libraries:** Pandas (Data Cleaning), Matplotlib (Visualization)
- **Environment:** JupyterLab

## Key Findings
Based on the analysis, the **United States** leads the world with 230 airlines, followed by Russia and Canada. Most top countries are major global economies with vast territories.

## How to Run
1. Download the `airlines.csv` and `.ipynb` files.
2. Run the notebook in JupyterLab or Anaconda.
3. Ensure the CSV file is in the same directory.bution
